TAC - How To Play this Game

  1. 1.) Type “start” to begin the game. 
  2. 2.) You will be given a description of the commands you can use in the game. There are several differnt games that you can play to improve your TAC skills. You can ask for a list of commands, if you need them.  
  3. 3.) You will usually have to connect with the need or feeling of the distressed citizen.

 

If you are guessing the need, then it should be a need from this list.

security, empathy, support, stability, safety, consideration, respect, understanding, shared reality, compassion, food/water, freedom, movement, choice, order, shelter, care, peace. 

If you are guessing a feeling, then it should be from the feelings list.

affectionate, confident, excited, inspired, grateful, peaceful, joyful, happy, confused, hurt, hungry, cold, worried, scared, suspicious, annoyed, frustrated, numb, shocked, embarrassed, ashamed, sad, depressed, tense, stressed out, vulnerable, fragile, shy, jealous 

 

You can always ask for the feelings list or needs list in the middle of the game if you want.
