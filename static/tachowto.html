TAC - How To Play this Game

This is a simple game where you will try to guess the needs of another person.

 

Game Play

  1. 1.) Type “start” to begin the game. 
  2. 2.) You will be prompted with a situation and a comment from a distressed citizen.  
  3. 3.) You will guess the need (or feeling) of the distressed citizen.
    1. If you guessed the correct need, then you can move to the next section.
    2. If you need to guess multiple correct answers you will continue the conversation until you have guessed enough correct answers.
    3. If you guess incorrectly, the system will continue the conversation

 

You will guess ONE need at a time. it will be a need from this list.

security, empathy, support, stability, safety, consideration, respect, understanding, shared reality, compassion, food/water, freedom, movement, choice, order, shelter, care, peace. 

 

The system is looking for keywords. It will treat the following responses identically, if you type:

            “Do you want safety”

            “Do you need safety dude?”

            “safety”
